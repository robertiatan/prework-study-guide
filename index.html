HTML

  • Head element contains imporant behind the scenes info about the webpage
    • The body element shows the content of the webpage to the viewer
  • HTML is the framework of a website

CSS

  • All HTML elements are made up of boxes
  • Margins indicate space around the outside of an element
  • Padding indicates spaces around the content from inside the element

Git

  • Git is version control, allowing devs to collaborate and share code
  • Git commands are used in GitBash
  • Commands like git status and git checkout are necessary to learn

JavaScript

  • A variable is a named container that allows us to store data in our code
    • Control flow is the order in which a computer executes code in a script.
